# World's Best CV Maker

A professional, ATS-friendly CV builder with an elegant dark theme and smooth animations.

## Features

- **ATS-Optimized** - Clean formatting that passes through Applicant Tracking Systems
- **Dark/Light Mode** - Beautiful bloom animation between themes
- **No Required Fields** - Fill in only what you want
- **Live Preview** - See your CV as you build it
- **PDF Export** - Download your CV instantly
- **Professional Design** - Elegant typography with Playfair Display & Inter fonts

## Usage

1. Open `index.html` in your browser
2. Fill in your information (all fields optional)
3. Click "Generate Preview" to see your CV
4. Click "Download PDF" to save

## Design

- Black & white aesthetic with white stroke buttons
- Dark mode by default
- Smooth theme transitions with circular bloom effect
- Fully responsive layout
